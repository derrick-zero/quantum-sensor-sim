Static calculateFieldAtPoint

  • calculateFieldAtPoint(sensor, point): Vector3

  • Calculates the magnetic field at a given point due to a single sensor using the point-charge approximation of the Biot–Savart law:

    B = (μ₀ / (4π)) * (q (v x r)) / |r|³

    Parameters

    • sensor: Sensor

      The sensor generating the field.

    • point: Vector3

      The evaluation point in space.

    Returns Vector3

    The static magnetic field vector at the specified point.

    Throws

    Error if the evaluation point coincides with the sensor's position.

Static calculateFieldAtPointTimeDependent

  • calculateFieldAtPointTimeDependent(sensor, point, time, frequency?): Vector3

  • Calculates a time-dependent magnetic field at a given point due to a sensor. The static magnetic field is modulated by a sinusoidal function of time.
